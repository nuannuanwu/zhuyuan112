记忆童年是什么?

记忆童年是宝宝校园生活的网络日志

你可以看到孩子在学校的一切动态，你可以了解教育知识，写上自己的留言。

记忆童年怎么用？

Step1   浏览

浏览丰富的成长内容，里面记载着众多宝宝的成长事迹和教育信息。

Step2   交流

对你感兴趣的内容进行收藏和评论，与更多宝宝家长交流。

Step3   互动

与老师沟通，给我们建议。互动中一起成长。

记忆童年客户端

记忆童年与您一起见证孩子的点滴每一天，随时随地的校园动态，科学专业的教育业信息。

  • 展示宝宝在校园的一切动态
  • 为宝宝建立成长档案
  • 发布专业教育信息
  • 提供家长与老师的交流平台
